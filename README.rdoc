# todo_pomodoro!

## overview

todo_pomodoro is todo list with the pomodoro workflow built in!

manage your todos, focus on one at a time for 25 minute intervals, and track your progress!

## pomodoro technique

pomodoro is an awesome workflow to get things done.

step 1: set a timer for 25 minutes
step 2: focus on one task for that time
step 3: get as far as you can
step 4: when the timer goes off take a 5 minute break
step 5: repeate.
